Introduction

Every website you visit is made using three basic languages: HTML, CSS, and (most of the time) JavaScript.

HTML is the structure of the page. CSS is the style of the page, and JavaScript handles the user interactions (ie - what happens if I click on this?

What We're Doing Today

We will construct a basic structure for our page using HTML and style the elements using CSS. We will not be using JavaScript today.

Setting Up Codepen

  1. Go to codepen.io and click "Sign Up" in the upper right-hand corner
  2. Once you're logged in, click the button that says "Create" and select "New Pen"
  3. If you'd like to use the view that Rachel showed during the demo, click "Change View" and select the first option
  4. Click the little gear icon near the CSS editor and select "Normalize" for the CSS base
  5. Click "Save & Close"

As you work through this tutorial, be sure to continue saving your work using the save button near the top.

If you get stuck or lost at any point, you can open up the examples that are embedded in the tutorial to look at the code, or you can click "Fork" to create a copy to use moving forward for your own.

Hero Section

The "hero section" of a website is the first thing that the user will see. It's usually a large banner image or pattern centered on the top of the page.

Hero Section HTML (Structure)

  1. Add a div and give it a class of "hero"
  2. Inside the div, add an h1 tag with the title of yourself, your website, or your product

Hero Section CSS (Style)

There are many different CSS properties that we'll use to style this hero section. First, click on your CSS tab and copy over the code below:


.hero {
  display: flex; 
  background-image: url("");
  background-size: cover;
  background-position: center center;
  height: 80vh;
  justify-content: center;
  align-items: center;
  color: white;
  text-shadow: black 4px 4px 20px;
  font-size: 24px;
}

(Want a detailed explanation of each property? Click here.)

Things to Tinker With

  1. Change background-position: center center; to background-position: bottom center; or background-position: top center;
  2. Change height: 80vh; to height: 30vh; or height: 50vh;
  3. Change align-items: center; to align-items: start; or align-items: end;
  4. Change the color to something other than white. Find a list of web-safe colors here.
  5. Change the number of pixels and color of text-shadow to see what happens.

Three Column Layout for Features

Next, we'll add text content below our hero section. Ultimately, we want three columns, each containing a small heading for a feature and a paragraph of text. Since the heading and text belong together, we'll group each of them with a div, then style that div.

Three Column Layout HTML (Structure)

  1. Create a div that has a class of "container"
  2. Inside of the container div, create another div that has the class of "features"
  3. Inside of features, create *three* separate divs that have the classes of feature-one, feature-two, and feature-three
  4. For each feature, add a small heading (like an h3) and a paragraph (p)

This is an example of what you should have so far:

See the Pen Three Column HTML by Rachel Warbelow (@rwarbelow) on CodePen.

Three Column Layout CSS (Style)

We'll again use display: flex; to make it so that our individual features are lined up on the page and flexing to take up space evenly.

First, we're going to make our features div display flex, in addition to giving it a property that we'll see the effect of later. Click on your CSS tab and add the following below your already-written code:

.features {
  display: flex;
  justify-content: space-between;
}

.feature-one {
  flex-basis: 28%;
  padding: 15px;
  background-color: rgba(241,144,45, 0.2);
}

.feature-two {
  flex-basis: 28%;
  padding: 15px;
  background-color: rgba(141,59,61, 0.2);
}

.feature-three {
  flex-basis: 28%;
  padding: 15px;
  background-color: rgba(83,122,49, 0.2);
}

(Want a detailed explanation of each property? Click here.)

Things to Tinker With

  1. Change the percentage for flex-basis to a larger or smaller number.
  2. Change the number of pixels used for padding.
  3. Change the numbers for the RGBA color. Make sure the first three numbers are between 0 and 255, and the final number is between 0 and 1.

Footer

We'll make a very simple footer that includes an SVG icon and a link.

Footer HTML (Structure)

Scroll to the very bottom of your HTML, and find the last two closing div tags. They look like this:


    </div>
</div>

The first tag closes our features div, and the second one closes our container div. We want our footer to be outside of the features, but still inside the container.

Between those two closing div tags, do the following:

  1. Add a div and give it a class of "footer"
  2. Within the footer div, create another div with the class of "contact"
  3. Click on "assets", then click "icons", and click on an icon you want to represent contacting you. This will copy that code to your clipboard
  4. Paste what was copied into the contact div. It will look pretty messy, but it will work
  5. Directly under the SVG, create a link to email yourself, like this: 

<a href="mailto:rachel@example.com">Contact Me</a>

Be sure to change the email address to your own.

This is an example of what you should have so far:

See the Pen Footer HTML by Rachel Warbelow (@rwarbelow) on CodePen.

Footer CSS (Style)

We want to center our icon and link, in addition to removing the default underline from our link. First, in your CSS tab on CodePen, create two containers for the style of our footer and the style of our link:


.footer {
  margin-top: 20px;
  display: flex;
  justify-content: space-around;
  margin-bottom: 25px;
}


a {
  text-decoration: none;
  color: rgba(83,122,49, 1);
}

Things to Tinker With

  1. Change the number of pixels for margin-top and margin-bottom.
  2. Change the text-decoration to line-through or overline.
  3. Change the numbers for the RGBA color. Make sure the first three numbers are between 0 and 255, and the final number is between 0 and 1.

Hosting Your Website

Now that you have your project built, let's upload it to the web. To do this, we'll use a service called Forge.

  1. Go to getforge.com and sign up for a free account. You'll need to confirm your email address.
  2. From your codepen, click the little button in the lower right-hand corner that says "Export" and select "Export .zip"
  3. Go back to Forge, choose a name for your first site (note, you can only change the first part -- you have to keep getforge.io on the end of it).
  4. Upload your zipped CodePen files
  5. Tada! Visit your custom link to see your site live. You can share this link with the world!
